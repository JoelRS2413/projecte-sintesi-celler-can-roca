# Proyecto de Síntesis
## Infraestructura informática del restaurante El Celler de Can Roca

Este proyecto consiste en diseñar la infraestructura informática para el restaurante **El Celler de Can Roca**.

## Servicios implementados

### Windows Server
- Active Directory
- DHCP
- DNS

### Ubuntu Server
- Servidor web
- MySQL
- Proxy
- Correo

## Infraestructura de red
- VLAN clientes
- VLAN empleados
- VLAN cámaras
